Programme de gestion d'afficheur GliderScore
--------------------------------------------
Le but de ce dossier est de fournir les elements necessaires pour la réalisation d'un afficheur a brancher en sortie d'un PC avec GliderScore.

3 modules :
- 1 module de transmission en 2.4Ghz Canal 0 avec display Oled - Receive_Big
- 1 module de reception en 2.4 Ghz Canal 0 pour gestion GRAND afficheur segment diplay 7 digit - Transmit_all
_ 1 module de recpetion en 2.4 Ghz Canal 0 sur chrono avec afficheur Nokia 5119 - Small_Chrono


le Harware est basé sur des modules arduino nano + nrf24l01 + Large Driver 7 digit 

https://www.sparkfun.com/products/13279
